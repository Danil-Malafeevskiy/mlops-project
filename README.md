# Fraud Detection MLOps Pipeline

## Описание
Проект реализует полный MLOps-цикл для классификации мошеннических транзакций по датасету Credit Card Fraud Detection.

## Цель бизнеса
Минимизировать потери от мошеннических операций за счет раннего обнаружения подозрительных транзакций.

## Цель ML
Построить модель, которая максимально хорошо находит мошеннические транзакции в условиях сильного дисбаланса классов

## Ключевые метрики
- Recall для класса Fraud — основная метрика (Пропустить мошенническую транзакцию хуже, чем ошибочно пометить честную как подозрительную)
- F1-score для класса Fraud
- PR-AUC — дополнительная метрика
- ROC-AUC — вспомогательная метрика
